## 第四阶段：AI辅助批量生成

### 使用 Ollama 批量生成新的人格描述+对话样本

```bash
# 准备 prompt template
python scripts/augment_personas.py --model llama3.2:3b
```

**Prompt 模板思路：**
1. 给定基础人格信息（traits, style_tags）
2. 要求生成 8-12 条符合该人格的对话样本
3. 每条对话要体现不同的场景和情绪
4. 避免重复和模板化表达

## 如何合并到新数据

```bash
python scripts/merge_enhanced.py \
  --base data/enhanced_base_personas.json \
  --target data/full_personas.json \
  --output data/full_personas_v2.json
```

合并后：
- 基础人格被强化
- 变体数量保持不变
