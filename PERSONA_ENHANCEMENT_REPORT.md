## 🔬 技术亮点

1. **多场景对话体系**：日常闲聊 / 情感安慰 / 尖锐辩论 / 哲学思考 / 神秘预言
2. **Ollama AI辅助**：批量生成符合人物性格的高质量中文对话
3. **morph方向支持**：每个人格都有可调节的性格变化方向
4. **数据向后兼容**：新结构不影响现有引擎的 embed/train/morph 流程
5. **可扩展架构**：后续只需追加 JSON 即可扩展更多人格
